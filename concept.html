Phase 02
Process
Sensor Measurement
Voltage & current sampled at high frequency per outlet
ESP32 Microcontroller
Dual-core processor handles all real-time computation
Power Computation
Calculates wattage (W) and energy consumed (kWh)
Cost Estimation
Multiplies kWh usage by electricity rate in real time
Cloud Transmission
Data pushed to cloud server over Wi-Fi via MQTT/HTTP
Mobile App Retrieval
App fetches and renders data from cloud in real time
